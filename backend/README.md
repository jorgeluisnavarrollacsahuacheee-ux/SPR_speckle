## Descripción del proyecto
El proyecto SPR Speckle Analyzer es una aplicación web académica para el análisis de patrones de speckle en imágenes. Procesa muestras, aplica transformaciones espaciales y de frecuencia, calcula métricas (FFT, contraste, intensidad media) y visualiza resultados en tiempo real con una interfaz basada en Material Design.

## Tecnologías utilizadas
- **Backend:** FastAPI, SQLAlchemy, PostgreSQL, OpenCV, NumPy
- **Frontend:** Vite + React, TailwindCSS, Material UI (MUI)
- **Base de datos:** PostgreSQL
- **Diseño:** Figma, MUI
